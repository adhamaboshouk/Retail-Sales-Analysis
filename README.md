📊 Retail Sales Analysis Project
📌 Project Overview

This project is a Retail Sales Analysis Dashboard built using SQL, Excel, and Power BI.
The goal is to analyze sales performance, understand customer behavior, and provide business insights that help in decision-making.

🎯 Objectives
Analyze overall sales performance
Identify top-selling products and categories
Track revenue trends over time
Understand customer purchasing behavior
Create an interactive dashboard for decision-making
🛠️ Tools & Technologies
SQL → Data cleaning, transformation, and querying
Excel → Initial data exploration and preprocessing
Power BI → Data visualization and dashboard creation
📂 Project Workflow
1. Data Collection

Raw retail sales dataset was imported into Excel/SQL.

2. Data Cleaning (Excel + SQL)
Removed duplicates
Handled missing values
Standardized formats (dates, categories, etc.)
3. Data Analysis (SQL)
Aggregated sales by product, region, and time
Created key metrics like:
Total Sales
Total Profit
Quantity Sold
Average Order Value
4. Visualization (Power BI)

Created an interactive dashboard including:

Sales trend over time 📈
Top products 🏆
Category performance
Regional analysis 🌍
KPI cards (Revenue, Profit, Orders)
📊 Key Insights
Certain products generate most of the revenue (Pareto pattern)
Sales vary significantly by month/season
Some regions outperform others in profitability
Discounts impact total profit margins
